# Personal Portfolio Website

This is a responsive, multi-page portfolio website built for my academic project. It showcases my technical skills, academic qualifications, and projects.

**Live Website:** [Click here to view my portfolio](https://Leoo197.github.io/Portfolio-website)

## Website Structure
The website is structured into the following separate HTML pages to keep content organized:

* `index.html`: The main landing page featuring my introduction, bio, and social links.
* `qualifications.html`: Details my academic journey (B-Tech, 12th, and 10th grades).
* `skills.html`: Highlights my technical toolkit and hobbies.
* `projects.html`: Showcases the software and web development projects I have built.
* `contact.html`: A clean form setup for visitors to reach out.

## Technologies Used
* **HTML5:** For the structural markup across all pages.
* **CSS3:** For styling, including Flexbox for layouts and media queries for mobile responsiveness.
